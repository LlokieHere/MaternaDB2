# MATERNADB: Integrated Maternal Health Information System

**MATERNADB** is a desktop-based information system designed to streamline the management of maternal healthcare data. It provides healthcare staff with an organized and efficient way to handle patient records, prenatal care monitoring, and appointment scheduling within a unified platform.

## 📌 Overview

The system centralizes critical patient information, allowing healthcare providers to easily access, update, and manage maternal health data. It is built to improve workflow efficiency, reduce manual record-keeping, and enhance patient care delivery.

## ⚙️ Key Features

* **Patient Records Management**
  Store and manage comprehensive patient information including personal details and medical history.

* **Prenatal Care Tracking**
  Monitor patient progress throughout pregnancy, including checkups and prescriptions.

* **Appointment Scheduling**
  Organize and manage patient appointments efficiently.

* **Prescription Management**
  Record and review prescribed medications and treatment details.

* **Dashboard Overview**
  View key statistics such as total number of patients and system activity.

## 🖥️ Built With

* **Python**
* **PyQt6** (Graphical User Interface)
* **POSTGRESQL / Database System** 

## 🎯 Purpose

This system is developed to support healthcare providers in delivering better maternal care by digitizing and organizing patient-related processes into a single, easy-to-use application.
